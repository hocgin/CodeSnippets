### 安装
> 默认将快捷指令安装到 ~/Library/Developer/Xcode/UserData/CodeSnippets/ 目录，执行以下命令后重启 Xcode 即可生效。

```shell
# 直接从 GitHub Raw 安装
curl -fsSL https://raw.githubusercontent.com/hocgin/CodeSnippets/main/install.sh | bash

# 或者在仓库目录里执行
./install.sh

# 也可以显式指定仓库目录
bash /path/to/install.sh /path/to/CodeSnippets
```
